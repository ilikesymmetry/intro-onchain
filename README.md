# Intro to Onchain

This is a [Next.js](https://nextjs.org) project bootstrapped with `npm create onchain`.


## Getting Started

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
